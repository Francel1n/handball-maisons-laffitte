# Application de gestion de présence pour le club de Handball de Maisons-Laffitte

Cette application permet aux joueurs du club de handball de Maisons-Laffitte de consulter les entraînements et aux administrateurs de gérer les différents évenements de la section senior.

Cette application naquis lors autour d'une bière au 46, elle a pour vocation de remplacer Team Pulse, très impopulaire auprès des joueurs notament pour ses publicités. Les fonctionalités principales seront assurée dans ce projet

## Fonctionnalités

- Sélection d'identité de joueur
- Consultation des entraînements
- 'Cochage' de présence ou non présence pour chaque entraînement
- Visualisation des présences
- Gestions des évenements pour les administrateurs

## Technologies utilisées

- Reactjs
- TypeScript
- Tailwind CSS
- Vite

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/francel1n/handball-maisons-laffitte.git
cd handball-maisons-laffitte

# Installer les dépendances
npm i

# Lancer l'application en mode développement
npm run dev
```

## Structure du projet

- `/src` - Code source de l'application
  - `/components` - Composants React réutilisables
  - `/lib` - Utilitaires et fonctions partagées
